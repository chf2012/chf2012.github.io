快捷键

Atom是GitHub推出的一款编辑器，该编辑器暂时还不是很成熟，比如目前还无法支持代码补全等功能。但是GitHub作为一个极具影响力的开源社区，Atom必然会得到快速发展。在此记录Atom的部分常用快捷键。

这些快捷键都可以在Atom的在线文档中查阅到。

PS: 其实所有快捷键都可以通过触发Ctrl + Shift + P命令而弹出的下拉框来查看。理论上来说只需记住该快捷键即可。
