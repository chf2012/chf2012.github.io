4.2 显示数量-limit()

LIMIT

  • LIMIT n1 # 返回前n1行，并且MySQL是从0开始的

  • LIMIT n1,n2 # 返回从第n1+1行(MySQL从0开始)，之后的n2行

  • LIMIT n1,n1 # 返回特定行的值

Limit结合排序Order by 往往有不错的效果
